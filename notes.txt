# 1️⃣ What is Git?

Git is a **Distributed Version Control System (DVCS)** used to track changes in files and collaborate with other developers.

## Why Use Git?

✅ Tracks file changes

✅ Maintains project history

✅ Supports teamwork

✅ Allows rollback to previous versions

✅ Enables branching and merging

---

# 2️⃣ What is GitHub?

GitHub is a cloud platform that hosts Git repositories online.

## Workflow

```text
Developer
    ↓ Push
 GitHub
    ↑ Pull
Developer
```

## Benefits

* Code Backup
* Team Collaboration
* Open Source Contributions
* Project Hosting

# 3️⃣ Git Repository

A repository (repo) is a project managed by Git.

## Initialize Repository

```bash
git init
```

### Example

```bash
mkdir Developer-Main
cd Developer-Main
git init
```

Output:

```text
Initialized empty Git repository
```

---

# 4️⃣ Git Status

Displays the current state of the repository.

## Command

```bash
git status
```

## Example Output

```text
On branch main
nothing to commit, working tree clean
```

## Uses

* Check modified files
* Check staged files
* Check current branch


# 8️⃣ Git Remote

Connects local repository with GitHub.

## Add Remote

```bash
git remote add origin <repository-url>
```

### Example

```bash
git remote add origin https://github.com/username/Git-Tutorial.git
```

## Verify Remote

```bash
git remote -v
```

# 1️⃣3️⃣ Git Branching

## What is a Branch?

A branch is an independent line of development.

### Main Branch

```text
main
 |
 A
```

### Feature Branch

```text
main
 |
 A

feature
 |
 A
```

---

## Create Branch

```bash
git branch feature
```

## Switch Branch

```bash
git switch feature
```

## Create + Switch

```bash
git switch -c feature
```

## View Branches

```bash
git branch
```

Example:

```text
* main
  feature
```

---

# 1️⃣4️⃣ Git Merge

Combines changes from one branch into another.

## Merge Feature Into Main

```bash
git switch main
git merge feature
```

### Before Merge

```text
main
 A

feature
 A → B
```

### After Merge

```text
main
 A → B

feature
 A → B
```

# 1️⃣5️⃣ Local Branch vs Remote Branch

## Local Branch

Exists only on your computer.

```text
Developer-Main
└── feature
```

## Push Branch to GitHub

```bash
git push -u origin feature
```

## Remote Branch

Exists on GitHub.

```text
GitHub
├── main
└── feature
```

# ⚠️ Common Errors & Solutions

## Error 1

```text
src refspec master does not match any
```

### Cause

Trying to push `master` while current branch is `main`.

### Solution

```bash
git push origin main
```

---

## Error 2

```text
remote contains work that you do not have locally
```

### Cause

GitHub has commits missing locally.

### Solution

```bash
git pull origin main
git push origin main
```

---

## Error 3

```text
fatal: not a git repository
```

### Cause

Not inside a Git repository.

### Solution

```bash
cd project-folder
git status
```
